
# THAPI CLI - Nestjs projects DDD

Thapi CLI to quickly build applications using hexagonal architecture, DDD and SOLID principles.


## Installation

Install CLI with npm

```bash
  1. Clone this repository
  2. Go to folder of this project.
  3. npm install -g .
```
