## Install CLI for create project of nestjs with hexagonal architecture

1. Download this repository
2. Go to folder of this project.
3. npm install -g .


## Usage
# For create App:
    thapi --new --name="APPLICATION_NAME"

# For create Entity:
    thapi --entity="ENTITY_NAME"
